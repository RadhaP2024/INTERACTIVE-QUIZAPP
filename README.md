Description

A dynamic quiz application that tests users' knowledge with multiple-choice questions. Features instant feedback, score tracking, and a clean, responsive interface.
Features

    ✅ Multiple choice questions with radio buttons

    ✅ Instant feedback on answers (correct/incorrect)

    ✅ Score tracking throughout the quiz

    ✅ Visual feedback with color-coded responses

    ✅ Progress through questions automatically

    ✅ Final score summary

Technologies Used

    HTML5

    CSS3 (Flexbox, Gradients, Animations)

    Vanilla JavaScript (DOM manipulation, Event handling)

Learning Outcomes

    DOM manipulation

    Event listeners and handlers

    Arrays and objects

    Conditional logic

    Dynamic content rendering
